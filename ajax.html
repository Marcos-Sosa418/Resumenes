¿Qué es AJAX?

AJAX (Asynchronous JavaScript and XML) es una técnica que permite que las aplicaciones web envíen y reciban datos del servidor sin recargar toda la página. Esto hace que las páginas sean más rápidas, interactivas y dinámicas.

Características de AJAX

  • Asíncrono: La página no se recarga, las solicitudes ocurren en segundo plano.
  • Interactividad: Permite actualizar solo una parte de la página.
  • Compatible con varios formatos: JSON, XML, HTML y texto plano.
  • Uso de tecnologías estándar: HTML, CSS, JavaScript y XMLHttpRequest.

Usos de AJAX

  • Actualizar contenido dinámicamente.
  • Validar formularios sin recargar la página.
  • Cargar datos en segundo plano.
  • Aplicaciones en tiempo real como chats o notificaciones.

Ejemplo de Funcionamiento de AJAX

A continuación, un ejemplo sencillo que carga datos desde un archivo usando AJAX:

<!DOCTYPE html>
<html>
<head>
    <title>Ejemplo de AJAX</title>
    <script>
        function loadData() {
            var xhr = new XMLHttpRequest();
            xhr.onreadystatechange = function() {
                if (xhr.readyState == 4 && xhr.status == 200) {
                    document.getElementById("data").innerHTML = xhr.responseText;
                }
            };
            xhr.open("GET", "data.txt", true);
            xhr.send();
        }
    </script>
</head>
<body>
    <h1>Ejemplo de AJAX</h1>
    <button type="button" onclick="loadData()">Cargar Datos</button>
    <div id="data">Los datos se cargarán aquí...</div>
</body>
</html>

Archivo data.txt

Estos son los datos cargados desde el servidor.

Descripción del Ejemplo

  • HTML + JavaScript: Contiene la función loadData() encargada de hacer la solicitud AJAX.
  • XMLHttpRequest: Objeto usado para enviar la solicitud al servidor.
  • onreadystatechange: Detecta cuando la respuesta está lista.
  • Botón: Llama a la función y actualiza el contenido dinámicamente.

Ventajas de Usar AJAX

  • Mejora la experiencia del usuario.
  • Evita recargar la página completa.
  • Permite crear aplicaciones web dinámicas y fluidas.
