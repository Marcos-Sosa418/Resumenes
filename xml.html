¿Qué es XML?

XML es un lenguaje de marcado diseñado para almacenar datos y describir su estructura. Es extensible, lo que significa que tú defines tus propias etiquetas. Fue creado para ser legible tanto para humanos como para máquinas.

Características principales

  • Extensible: Puedes crear tus propias etiquetas.
  • Estructurado: Representa datos de forma jerárquica.
  • Legible: Fácil de comprender por humanos.
  • Interoperable: Compatible con muchos lenguajes y tecnologías.
  • Basado en texto: Fácil de transportar y almacenar.

Ejemplo grande: Menú en XML (con su equivalente JSON)

XML

<menu id="file" value="File">
  <popup>
    <menuitem value="New" onclick="CreateNewDoc()" />
    <menuitem value="Open" onclick="OpenDoc()" />
    <menuitem value="Close" onclick="CloseDoc()" />
  </popup>
</menu>

JSON equivalente

{
  "menu": {
    "id": "file",
    "value": "File",
    "popup": {
      "menuitems": [
        { "value": "New", "onclick": "CreateNewDoc()" },
        { "value": "Open", "onclick": "OpenDoc()" },
        { "value": "Close", "onclick": "CloseDoc()" }
      ]
    }
  }
}

Comparación XML vs JSON

Característica XML JSON
Sintaxis Basada en etiquetas Basada en objetos
Tamaño Más verboso Más compacto
Legibilidad Buena Muy buena
Uso común Documentos, configuración, datos complejos APIs, datos web, microservicios
